Description

The Corsi-Block Tapping task is a visual-spatial memory assessment. In this game, you must memorize a series of blocks that have been tapped by the computer and then you must tap the blocks in the same order. You must learn a succession of flashing blocks while playing the game. There are several rounds of testing, and each round adds one more block to remember. If you fail to tap the blocks in the correct flashing order, you will be asked to repeat the game. The test is over if you don't pass the second time. You can see your personal record for the highest number of memorized blocks.

Play as many times as you like and try to beat the high score!
